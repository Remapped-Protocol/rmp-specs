# Extrinsics (spec)

This document describes intended extrinsics at a high level.
Names and parameters may evolve before implementation is published.

## Messaging-related
- submit_attestation(...)
  - Writes an on-chain attestation for a signed message (no content stored)

## Identity-related
- register_identity(...)
- update_identity(...)
- attest_identity(...)

## Governance-related (non-token)
- constrained administrative actions as defined by governance model
